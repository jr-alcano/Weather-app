1. To-Do List Application

Description:
A web app that allows users to create, edit, and delete tasks. Tasks can be marked as completed, and completed tasks and pending tasks are 
distinguished from each other visually.

Features:

    Add, edit, and delete tasks.
    Mark tasks as complete or incomplete.
    Filter tasks by their completion status.
    Store tasks locally in the browser using localStorage or a database.

2. Recipe Finder

Description:
A basic application that allows users to search for recipes based on ingredients or keywords using some API.

Features:

    Search for recipes by keyword.
    Display recipe details, including ingredients and steps.
    Save favorite recipes locally or in a database.

3. Notes App

Description:
A minimalist app for taking and organizing notes. Users can create, edit, and delete notes with basic formatting options.

Features:

    Create, edit, and delete notes.
    Organize notes by categories or tags.
    Save notes locally or to a database.

4. Weather Dashboard (Current Project Example)

Description:
A dashboard to search for weather data by city, with options to save favorite cities and display their weather details.

Features:

    Search for current weather by city.
    Save favorite cities and view their weather data.
    Use a weather API for data.
    Display temp and a general weather description.
